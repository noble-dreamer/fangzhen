# V1 粗图的物理机制

V1 的目标不是直接求出真实壁厚损失，而是把频域测量矩阵 `H(tx, rx, f)` 变成一个和缺陷空间位置有关的粗先验图。

## 1. 输入物理量

频域仿真导出的是复响应：

```text
H(tx, rx, f) = receiver patch weighted radial displacement
```

健康样本为 `H0`，损伤样本为 `Hd`。V1 不直接使用 `abs(Hd)`，而是使用健康-损伤差异：

```text
DeltaH    = Hd - H0
RelDeltaH = abs(Hd - H0) / (abs(H0) + eps)
LogLoss   = log(abs(H0)+eps) - log(abs(Hd)+eps)
PhaseDiff = angle(Hd * conj(H0))
```

这样做的原因是单个 `abs(Hd)` 主要包含入射场、传播损耗、传感器耦合和几何路径差异，而缺陷信息主要体现在相对健康状态的变化中。

## 2. 展开管壁路径假设

管道外表面展开为二维平面：

```text
x = Rm * theta
y = z
```

对每个 `tx-rx`，导波在展开平面上近似沿直线传播。考虑管道周向周期性后，同一对传感器有多个螺旋阶次路径：

```text
m = -1, 0, 1
```

V1 认为：如果某条 `tx-rx` 路径附近有缺陷，该路径对应的频域响应差异会变大。因此可以把该路径的异常值反投影回展开平面。

## 3. Ray-tube 反投影

V1 不是把一条路径画成无限细直线，而是画成有宽度的 tube kernel：

```text
K(p | tx, rx, m) = exp(-d_perp(p, ray)^2 / (2*sigma_ray^2)) / L_ray
```

其中：

- `p` 是展开图中的候选像素；
- `d_perp` 是像素到螺旋路径直线的垂直距离；
- `sigma_ray` 是路径宽度；
- `L_ray` 是路径长度，用于几何归一化。

每个频点、每个 tx-rx、每个螺旋阶次都会给路径附近像素加权累积。最后除以覆盖度：

```text
pic(p) = sum(g(tx,rx,f) * K(p)) / (sum(K(p)) + eps)
```

这里 `g` 可以是 `RelDeltaH`、`LogLoss` 或 `PhaseDiff`。

## 4. 为什么会出现条带或拖尾

V1 本质是弱先验层析图，不是全波形反演。

单条路径只能说明“这条路径附近可能有异常”，不能确定缺陷在路径上的具体位置。只有多角度、多发射、多接收、多频率的异常路径在同一区域交汇时，缺陷位置才会被增强。

因此 V1 常见图像特征是：

- 缺陷附近变亮；
- 同时存在沿路径方向的条带；
- 缺陷形状比 label 更模糊；
- 多缺陷样本中不同缺陷可能互相产生交叉伪影。

这些现象是 V1 模型的正常限制，不代表代码一定错误。

## 5. V1 的作用边界

V1 适合：

- 给 diffusion 提供缺陷可能位置；
- 比较不同频点是否对定位有帮助；
- 快速筛选异常路径和频段；
- 作为后续 V2/V3 的初始粗图。

V1 不适合：

- 精确恢复缺陷边界；
- 精确预测最大深度；
- 单独作为最终缺陷图；
- 解释强多模态散射和复杂相位干涉。

因此评价 V1 时，不应只看像素级完全重合，而应同时看：

- 高响应区域是否接近 label；
- 异常中心是否靠近缺陷中心；
- 多个通道是否在同一区域一致变亮；
- 覆盖度是否足够。
